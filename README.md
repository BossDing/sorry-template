# node-sorry

`sorry有钱真的可以为所欲为`

## 说明

思路参考[sorry](https://github.com/xtyxtyx/sorry)和[nodejs版](https://github.com/q809198545/node-sorry)；
前面的nodejs版用的是express写的，这里写了个koa2版本的

## 技术栈
前端： vue iview UI 组件库
后端：nodejs koa2

## 使用
dist里提供了前端模板，可以直接进入server目录开启项目
先安装依赖
```
cd server
npm install 

```

运行

```
npm run start

```

用浏览器打开：http://127.0.0.1:8888
